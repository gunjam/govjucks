# Govjucks

Govjucks is a fork of [Nunjucks](https://mozilla.github.io/nunjucks/), a full
featured templating engine for javascript. It is heavily inspired by
[jinja2](http://jinja.pocoo.org/).

## Installation

`npm install govjucks`

To use the file watcher built-in to Govjucks, Chokidar must be installed
separately.

`npm install govjucks chokidar`
